# Everystone

Everystone is a trusted infrastructure layer for certified gemstones, especially diamonds.

**"Deposit once. Trade many times."**

A gemstone is deposited once into a trusted vault or verified custody system. After that, ownership can be transferred digitally without physically moving the stone every time.

This project is the initial landing page designed to test market demand and collect qualified leads from five key customer segments:
1. Diamond dealers and traders
2. Jewelry brands and manufacturers
3. Private gemstone owners and sellers
4. Investors and alternative asset buyers
5. Vaults, labs, custodians, and certification partners

The design direction is fintech + luxury + asset infrastructure: premium, institutional, trustworthy, minimal, and clear.

## Tech Stack
- Flutter Web (Adaptive/Responsive)
- `google_fonts` for typography (Cinzel / Inter)

## Setup and Run
To run the project locally:
```bash
flutter pub get
flutter run -d web
```
